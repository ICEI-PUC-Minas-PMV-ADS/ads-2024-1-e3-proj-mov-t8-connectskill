| **Caso de Teste** 	| **CT-02 – Busca por grupos/interesses ** 	|
|:---:	|:---:	|
|	Requisito Associado 	| RF-002 |
| Objetivo do Teste 	| Verificar se o usuário consegue realizar busca por grupos/interesses. |
